jPop Up v.0.1

Author: Tony Bispo Pinheiro aka Tony Pine

Usage:

$('.container').jPopup( params );

Params (default):
{
linkClass: '.showpop',
popupClass: '.jPopup',
popEl: false,
event: 'mouseover',
top: 25,
left: 0,
delayTime: 2000,
}

The activeClass is the class for styling the placeholder text.

Requirements:

JQuery and Modernizr.

Example:

Hover:

$('.jPopup-container').jPopup();
